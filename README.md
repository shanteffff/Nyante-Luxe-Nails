# Nyante Luxe Nails

## Project Description
Nyante Luxe Nails is an e-commerce website developed as a group project for CIT2011. The website allows users to register, log in, browse products, add items to their shopping cart, complete checkout, and generate invoices.

## Features
- User Registration
- User Login
- Product Catalog
- Shopping Cart
- Checkout
- Invoice Generation
- User Frequency Report

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub

## How to Run the Project
1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Register a user and explore the website.
